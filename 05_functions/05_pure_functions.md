# ✨ Чистые функции и побочные эффекты
```toc
minLevel: 2
```

## Что такое чистая функция?

**Чистая функция** (pure function) — это функция, которая:
1. **Детерминированная** — всегда возвращает одинаковый результат для одинаковых аргументов
2. **Без побочных эффектов** — не изменяет внешнее состояние программы

```python
# ✅ Чистая функция
def add(a, b):
    return a + b

print(add(2, 3))  # 5
print(add(2, 3))  # 5 — всегда одинаковый результат
```

## Что такое побочные эффекты?

**Побочный эффект** (side effect) — это любое изменение состояния вне функции:

- Изменение глобальных переменных
- Изменение переданных аргументов (для изменяемых типов)
- Вывод на экран (`print`)
- Запись в файл
- Отправка данных по сети
- Генерация случайных чисел
- Получение текущего времени

```python
# ❌ Нечистая функция — изменяет глобальную переменную
counter = 0

def increment():
    global counter
    counter += 1
    return counter

print(increment())  # 1
print(increment())  # 2 — результат зависит от состояния!
```

## 💡 Преимущества чистых функций

1. **Предсказуемость** — легко понять, что делает функция
2. **Легкое тестирование** — не нужно настраивать окружение
3. **Переиспользование** — работает в любом контексте
4. **Параллельное выполнение** — безопасно запускать одновременно
5. **Кэширование** — можно сохранять результаты

## Примеры чистых функций

```python
# ✅ Чистая — зависит только от аргументов
def calculate_price(price, quantity, discount=0):
    """Вычисляет стоимость с учётом скидки"""
    cost = price * quantity
    discounted = cost * (1 - discount / 100)
    return discounted

# ✅ Чистая — создаёт новый список
def double_list(numbers):
    """Удваивает элементы списка"""
    return [x * 2 for x in numbers]

# ✅ Чистая — работает только с аргументами  
def format_price(price, currency="₽"):
    """Форматирует цену"""
    return f"{price:,.2f} {currency}".replace(",", " ")
```

## Примеры нечистых функций

```python
# ❌ Нечистая — выводит на экран
def print_sum(a, b):
    print(a + b)  # Побочный эффект!

# ❌ Нечистая — изменяет список
def double_in_place(numbers):
    for i in range(len(numbers)):
        numbers[i] *= 2
    return numbers

# ❌ Нечистая — использует случайные числа
import random
def get_random_number():
    return random.randint(1, 100)  # Недетерминированная!

# ❌ Нечистая — использует текущее время
from datetime import datetime
def get_greeting():
    hour = datetime.now().hour  # Зависит от времени!
    if hour < 12:
        return "Good morning!"
    return "Good afternoon!"
```

## 🔄 Рефакторинг: от нечистых к чистым

### Было (нечистая)

```python
results = []

def process_number(num):
    """Нечистая: изменяет глобальный список"""
    if num % 2 == 0:
        results.append(num * 2)

process_number(2)
process_number(3)
process_number(4)
print(results)  # [4, 8]
```

### Стало (чистая)

```python
def process_number(num):
    """Чистая: просто возвращает результат"""
    if num % 2 == 0:
        return num * 2
    return None

def process_numbers(numbers):
    """Чистая: обрабатывает список и возвращает новый"""
    results = []
    for num in numbers:
        processed = process_number(num)
        if processed is not None:
            results.append(processed)
    return results

numbers = [2, 3, 4]
results = process_numbers(numbers)
print(results)  # [4, 8]
```

## 🎯 Практический пример: архитектура программы

```python
# ✅ Чистые функции (логика)
def process_data(data):
    """Обрабатывает данные"""
    return [x * 2 for x in data if x > 0]

def format_result(data):
    """Форматирует результат"""
    return "\n".join(str(x) for x in data)

# ❌ Функции с побочными эффектами (ввод/вывод)
def read_file(filename):
    """Читает файл"""
    with open(filename, 'r') as f:
        return [int(line.strip()) for line in f]

def write_file(filename, content):
    """Записывает файл"""
    with open(filename, 'w') as f:
        f.write(content)

# Координация
def main():
    # Побочный эффект: чтение
    data = read_file('input.txt')
    
    # Чистая логика
    processed = process_data(data)
    formatted = format_result(processed)
    
    # Побочный эффект: запись
    write_file('output.txt', formatted)
    print("Done!")  # Побочный эффект: вывод

if __name__ == "__main__":
    main()
```

## Как писать чистые функции

### ✅ Рекомендации

1. **Используйте параметры** вместо глобальных переменных
2. **Возвращайте новые объекты** вместо изменения существующих
3. **Избегайте `print`** — лучше возвращайте данные
4. **Не используйте `random`** — принимайте seed как параметр
5. **Не используйте `datetime.now()`** — принимайте время как параметр

### ❌ Чего избегать

1. Изменения глобальных переменных
2. Изменения переданных аргументов
3. Вывода на экран
4. Чтения/записи файлов внутри логики
5. Использования `random.random()`, `datetime.now()` без параметров

## Резюме

**Чистая функция:**
- Результат зависит только от аргументов
- Нет побочных эффектов
- Всегда одинаковый результат для одинаковых аргументов

**Побочные эффекты:**
- Изменение внешнего состояния
- Вывод/ввод данных
- Недетерминированное поведение

**Преимущества чистых функций:**
- Предсказуемость
- Лёгкое тестирование
- Переиспользование
- Возможность параллельного выполнения

**Практика:**
- Стремитесь к чистым функциям для логики
- Изолируйте побочные эффекты
- Разделяйте вычисления и ввод/вывод
- Документируйте функции с побочными эффектами
