# 🧮 Операторы
```toc
minLevel: 2
```

Операторы — это символы, которые выполняют действия над значениями. Здесь собраны все основные операторы Python в одном месте.

## Арифметические операторы

```python
# Базовая арифметика
2 + 3      # 5   — сложение
7 - 2      # 5   — вычитание
3 * 4      # 12  — умножение
10 / 3     # 3.3333...  — деление (всегда возвращает float!)
10 // 3    # 3   — целочисленное деление (отбрасывает дробную часть)
10 % 3     # 1   — остаток от деления
2 ** 3     # 8   — возведение в степень

# Унарные операторы
x = 5
-x         # -5  — смена знака
+x         # 5   — без изменений (редко используется)
```

### Особенности деления

```python
>>> 10 / 2
5.0          # Даже если делится нацело — результат float

>>> 10 // 3
3            # Целочисленное деление: дробная часть отброшена

>>> -10 // 3
-4           # Округляет вниз, а не к нулю! (-3.33... → -4)

>>> 10 % 3
1            # Остаток: 10 = 3*3 + 1

>>> -10 % 3
2            # Остаток всегда неотрицательный при положительном делителе
```

### Составные операторы присваивания

Краткая запись для «изменить и связать обратно»:

```python
x = 10
x += 3     # x = x + 3  → 13
x -= 2     # x = x - 2  → 11
x *= 4     # x = x * 4  → 44
x /= 2     # x = x / 2  → 22.0
x //= 3    # x = x // 3 → 7.0
x %= 5     # x = x % 5  → 2.0
x **= 3    # x = x ** 3 → 8.0
```

## Операторы сравнения

Возвращают [[03_2_1_boolean|булево значение]] — `True` или `False`:

```python
5 == 5     # True  — равно
5 != 3     # True  — не равно
5 > 3      # True  — больше
5 < 3      # False — меньше
5 >= 5     # True  — больше или равно
5 <= 4     # False — меньше или равно
```

### Цепочки сравнений

В Python можно писать цепочки — как в математике:

```python
# Вместо: x > 0 and x < 10
x = 5
0 < x < 10     # True — x между 0 и 10

# Несколько сравнений
1 < 2 < 3 < 4  # True
1 < 2 > 0      # True (1 < 2 и 2 > 0)
```

## Логические операторы

```python
True and False   # False — И: оба должны быть True
True or False    # True  — ИЛИ: хотя бы один True
not True         # False — НЕ: инвертирует значение
```

### Практическое использование

```python
age = 25
has_license = True

# Проверка нескольких условий
can_drive = age >= 18 and has_license  # True

# Проверка альтернатив
is_weekend = day == "Saturday" or day == "Sunday"

# Отрицание
is_minor = not (age >= 18)  # False
```

## Оператор принадлежности `in`

Проверяет, содержится ли элемент в коллекции:

```python
# В строках
"py" in "python"       # True
"Java" in "python"     # False
"py" not in "Java"     # True

# В списках
3 in [1, 2, 3, 4]     # True
5 in [1, 2, 3, 4]     # False

# В словарях (проверяет ключи)
"name" in {"name": "Иван", "age": 25}   # True
"Иван" in {"name": "Иван", "age": 25}   # False (ищет среди ключей!)
```

## Операторы идентичности `is`

Проверяют, ссылаются ли два имени на **один и тот же объект** в памяти (не путать с `==`, который сравнивает **значения**):

```python
a = [1, 2, 3]
b = a             # Тот же объект
c = [1, 2, 3]     # Другой объект с таким же содержимым

a is b       # True  — один и тот же объект
a is c       # False — разные объекты
a == c       # True  — значения равны

a is not c   # True  — разные объекты
```

### Когда использовать `is`

```python
# ✅ Для сравнения с None — всегда is
x = None
if x is None:
    print("Значение не задано")

# ❌ Не используй is для сравнения чисел и строк
x = 1000
y = 1000
x is y    # Может быть True или False — зависит от реализации!
x == y    # True — вот так правильно
```

> 💡 Подробнее о `is` и `==` — в разделе [[03_2_2_none|None]].

## Приоритет операторов

От **высшего** к **низшему** (то, что выше, выполняется первым):

```python
# 1. Скобки                ()
# 2. Возведение в степень  **
# 3. Унарные               +x, -x, not
# 4. Умножение/деление     *, /, //, %
# 5. Сложение/вычитание    +, -
# 6. Сравнения             ==, !=, <, >, <=, >=, is, in
# 7. Логическое НЕ         not
# 8. Логическое И          and
# 9. Логическое ИЛИ        or
```

### Примеры приоритетов

```python
2 + 3 * 4       # 14, не 20 — умножение первым
(2 + 3) * 4     # 20 — скобки меняют порядок

2 ** 3 ** 2     # 512, не 64 — ** правоассоциативен: 2 ** (3 ** 2) = 2 ** 9

True or False and False   # True — and выполняется первым
(True or False) and False # False — скобки меняют порядок

not 5 > 3       # False — сначала 5 > 3 (True), потом not True
```

> 💡 Если сомневаешься в приоритете — **ставь скобки**. Явное лучше неявного.

## 🔎 Проверка на ходу

```python
# Что вернут эти выражения?
>>> 7 // 2
3

>>> 7 % 2
1

>>> -7 // 2
-4     # Не -3! Округление вниз

>>> 2 ** 3 ** 2
512    # 2 ** 9, не 8 ** 2

>>> "hello" * 3
'hellohellohello'

>>> 10 == 10.0
True   # Разные типы, но равные значения

>>> [] is []
False  # Два разных пустых списка — разные объекты
```

## 💡 Главное

```python
# Арифметика: + - * / // % **
# Сравнение:  == != < > <= >=
# Логика:     and or not
# Принадлежность: in, not in
# Идентичность:   is, is not

# / всегда возвращает float, даже 10 / 2 → 5.0
# // округляет вниз, не к нулю
# is сравнивает объекты, == сравнивает значения
# Если непонятен приоритет — ставь скобки
```
