## О чём этот раздел

В основном разделе ООП мы учились **пользоваться** механизмами Python: создавать классы, наследоваться, переопределять методы, применять `@property`, `Enum` и другие готовые инструменты.

В этом разделе мы разбираемся, **как эти механизмы работают изнутри** — и учимся расширять их под свои задачи.

| Основы ООП | Продвинутое ООП |
|---|---|
| Используем `@property` | Понимаем, как `@property` работает через дескрипторы |
| Используем наследование | Управляем процессом создания классов через метаклассы |
| Пишем классы | Пишем классы, которые генерируют другие классы |
| Используем `Enum` из библиотеки | Понимаем, как устроены перечисления и как их расширять |

Это уже не «как писать классы», а «как Python строит классы». Тон раздела другой — больше про внутреннее устройство языка и меньше про повседневные задачи.

## Нужно ли это изучать?

**Короткий ответ:** не сразу.

Темы этого раздела встречаются в коде фреймворков (Django, SQLAlchemy, Pydantic) и в продвинутых библиотеках — но в прикладном коде применяются редко. Сначала стоит уверенно освоить основы ООП, модули, исключения и написать несколько реальных программ. К этому разделу разумно возвращаться:

- когда читаешь исходники фреймворка и натыкаешься на `metaclass=` или дескрипторы
- когда проектируешь библиотеку, которая должна быть удобной в использовании
- когда готовишься к техническому собеседованию на более глубокую позицию
- из чистого любопытства — чтобы понять, как Python устроен

## 📝 Шпаргалка

### Дескриптор

```python
class Validator:
    def __set_name__(self, owner, name):
        self.name = f"_{name}"

    def __get__(self, obj, objtype=None):
        if obj is None:
            return self
        return getattr(obj, self.name)

    def __set__(self, obj, value):
        setattr(obj, self.name, value)
```

### Миксин

```python
class LogMixin:
    def log(self, message):
        print(f"[{self.__class__.__name__}] {message}")

class MyService(LogMixin, BaseService):
    pass
```

### Метакласс

```python
class Meta(type):
    def __new__(mcs, name, bases, namespace):
        cls = super().__new__(mcs, name, bases, namespace)
        return cls

class MyClass(metaclass=Meta):
    pass
```

### `Enum`

```python
from enum import Enum, auto

class Color(Enum):
    RED = auto()
    GREEN = auto()
    BLUE = auto()
```

### Singleton

```python
class Singleton:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
```
