## Зачем работать с файлами?

Программы часто работают с данными, которые нужно сохранить между запусками — конфигурации, логи, результаты вычислений, данные пользователей. Файлы — основной способ хранения таких данных.

## Открытие файлов: `open()` и `with`

Для работы с файлом его нужно **открыть**, выполнить операции и **закрыть**. Самый правильный способ — конструкция `with`:

```python
# ✅ Правильный способ — with гарантирует закрытие файла
with open("example.txt", "r") as f:
    content = f.read()
    print(content)
# Файл автоматически закрыт после выхода из блока with

# ❌ Ненадёжный способ — можно забыть закрыть
f = open("example.txt", "r")
content = f.read()
f.close()  # Если до этой строки возникнет ошибка — файл останется открытым
```

> 💡 Подробнее о том, как работает `with` — в разделе [[08_6_context_managers|Контекстные менеджеры]].

## Режимы открытия

```python
open("file.txt", "r")   # Чтение (по умолчанию). Файл должен существовать
open("file.txt", "w")   # Запись. Создаёт файл или ПЕРЕЗАПИСЫВАЕТ существующий!
open("file.txt", "a")   # Дополнение. Создаёт файл или добавляет в конец
open("file.txt", "x")   # Создание. Ошибка, если файл уже существует
```
