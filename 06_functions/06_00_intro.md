## 📝 Шпаргалка

### Базовый синтаксис

```python
def функция(параметр1, параметр2=значение):
    """Документация функции"""
    # Тело функции
    return результат
```

### Виды параметров

```python
def полная_функция(
    только_поз, /,              # Только позиционный
    обычный,                    # Позиционный или ключевой
    со_значением=10,            # Со значением по умолчанию
    *args,                      # Произвольное количество позиционных
    только_ключ,                # Только ключевой
    **kwargs                    # Произвольное количество ключевых
):
    pass
```

### Lambda

```python
lambda x: x ** 2
```

### Декоратор

```python
@декоратор
def функция():
    pass
```

### Рекурсия

```python
def рекурсия(n):
    if базовый_случай:
        return базовый_результат
    return рекурсия(изменённый_n)
```
