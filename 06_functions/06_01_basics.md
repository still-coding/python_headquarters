## Что такое функция?

**Функция** — это именованный блок кода, который выполняет определённую задачу. Функции помогают:
- **Избегать повторения кода** — написали один раз, используем много раз
- **Разбивать программу на части** — каждая функция решает свою задачу
- **Делать код понятнее** — функция с хорошим именем объясняет, что она делает

## Синтаксис определения функции

```python
def имя(список параметров):
    """Документация функции (необязательно)"""
    # Тело функции
    # ...
    return выражение  # необязательно
```

**Ключевые моменты:**
- `def` — ключевое слово для создания функции
- `имя` — имя, по которому мы будем вызывать функцию
- `список параметров` — входные данные (может быть пусто)
- `return` — возвращает результат (если нет `return`, функция возвращает `None`)

## Простой пример

```python
def greet():
    """Выводит приветствие"""
    print("Hello, world!")

# Вызов функции
greet()  # Выведет: Hello, world!
```

## Функция с параметрами

```python
def add(a, b):
    """Возвращает сумму двух чисел"""
    return a + b

# Вызов функции с аргументами
result = add(3, 4)
print(result)  # 7
```

**Важно:** 
- `a` и `b` — это **параметры** (указываются при определении функции)
- `3` и `4` — это **аргументы** (передаются при вызове функции)

## Функция как исполняемый код

В Python `def` — это **исполняемый оператор**, а не декларация:

```python
x = 0

if x:
    def my_function(a):
        return a * a
else:
    def my_function(a):
        return a + 2

print(my_function(5))  # 7 (потому что x == 0, выполнилась ветка else)
```

Это означает, что функция создаётся **во время выполнения программы**, а не заранее.

## Функция создаёт объект

Когда Python выполняет `def`, он:
1. Создаёт **объект-функцию**
2. **Связывает** его с именем

```python
def square(x):
    return x * x

print(type(square))  # <class 'function'>
print(square)        # <function square at 0x...>

# Функцию можно присвоить другой переменной
my_func = square
print(my_func(5))  # 25
```

## 🎯 Пример с реальной задачей

```python
def intersect(list1, list2):
    """
    Возвращает список элементов, которые есть в обоих списках.
    
    Параметры:
        list1: первый список
        list2: второй список
    
    Возвращает:
        список общих элементов
    """
    result = []
    for item in list1:
        if item in list2:
            result.append(item)
    return result

# Использование
a = [1, 2, 3, 4, 5]
b = [3, 4, 5, 6, 7]
common = intersect(a, b)
print(common)  # [3, 4, 5]
```

## 💡 Особенности Python-функций

**Python не требует:**
- Объявлять типы параметров
- Объявлять тип возвращаемого значения
- Указывать, сколько параметров будет

**Python автоматически:**
- Определяет типы во время выполнения
- Связывает аргументы с параметрами при вызове
- Возвращает `None`, если нет явного `return`

## Практические примеры

### Функция без возвращаемого значения

```python
def print_multiplication_table(number):
    """Выводит таблицу умножения для заданного числа"""
    for i in range(1, 11):
        print(f"{number} × {i} = {number * i}")

print_multiplication_table(5)
# 5 × 1 = 5
# 5 × 2 = 10
# ...
# 5 × 10 = 50
```

### Функция с условиями

```python
def absolute_value(number):
    """Возвращает абсолютное значение числа"""
    return -number if number < 0 else number

print(absolute_value(-10))  # 10
print(absolute_value(5))    # 5
```

### Несколько return

```python
def check_sign(number):
    """Возвращает описание знака числа"""
    if number > 0:
        return "positive"
    elif number < 0:
        return "negative"
    else:
        return "zero"

print(check_sign(5))   # positive
print(check_sign(-3))  # negative
print(check_sign(0))   # zero
```

## 🔎 Проверка на ходу

Интерактивный REPL (Read-Evaluate-Print Loop) режим — твой друг!
Используй его, чтобы быстро проверять гипотезы, делать промежуточные проверки и вычисления.

```python
>>> def double(x):
...     return x * 2
...
>>> double(5)
10
>>> type(double)
<class 'function'>
>>> double.__name__
'double'
```

## Резюме

- Функция — это блок кода с именем
- `def` создаёт функцию во время выполнения
- Параметры — это имена в определении функции
- Аргументы — это значения при вызове
- `return` возвращает результат
- Без `return` функция возвращает `None`
- Функции — это объекты, их можно присваивать переменным
