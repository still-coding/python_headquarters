### `pop()` - –£–¥–∞–ª–µ–Ω–∏–µ –∏ –≤–æ–∑–≤—Ä–∞—â–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ –ø–æ –∏–Ω–¥–µ–∫—Å—É

## 3. –ú–µ—Ç–æ–¥—ã –ø–æ–∏—Å–∫–∞ –∏ –ø–æ–¥—Å—á–µ—Ç–∞

### `index()` - –ü–æ–∏—Å–∫ –∏–Ω–¥–µ–∫—Å–∞ –ø–µ—Ä–≤–æ–≥–æ –≤—Ö–æ–∂–¥–µ–Ω–∏—è

```python
list.index(value: Any, start: int = 0, stop: int = sys.maxsize) -> int
```
- **`value`** - –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è –ø–æ–∏—Å–∫–∞
- **`start`** - –Ω–∞—á–∞–ª—å–Ω–∞—è –ø–æ–∑–∏—Ü–∏—è –ø–æ–∏—Å–∫–∞
- **`stop`** - –∫–æ–Ω–µ—á–Ω–∞—è –ø–æ–∑–∏—Ü–∏—è –ø–æ–∏—Å–∫–∞

```python
fruits = ["—è–±–ª–æ–∫–æ", "–±–∞–Ω–∞–Ω", "–≤–∏—à–Ω—è", "–±–∞–Ω–∞–Ω"]

print(index)  # 1 (–∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ –≤—Ö–æ–∂–¥–µ–Ω–∏—è)

# –ü–æ–∏—Å–∫ —Å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–π –ø–æ–∑–∏—Ü–∏–∏
index2 = fruits.index("–±–∞–Ω–∞–Ω", 2)  # –ò—â–µ–º –Ω–∞—á–∏–Ω–∞—è —Å –∏–Ω–¥–µ–∫—Å–∞ 2
print(index2)  # 3

# –û—à–∏–±–∫–∞, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç –Ω–µ –Ω–∞–π–¥–µ–Ω
try:
    index3 = fruits.index("–ª–∏–º–æ–Ω")  # ValueError: '–ª–∏–º–æ–Ω' is not in list
except ValueError as e:
    print(f"–û—à–∏–±–∫–∞: {e}")

# –ë–µ–∑–æ–ø–∞—Å–Ω—ã–π –ø–æ–∏—Å–∫
def safe_index(lst, value):
    try:
        return lst.index(value)
    except ValueError:
        return -1  # –ò–ª–∏ None, –µ—Å–ª–∏ –ø—Ä–µ–¥–ø–æ—á–∏—Ç–∞–µ—Ç–µ

fruits = ["—è–±–ª–æ–∫–æ", "–±–∞–Ω–∞–Ω", "–≤–∏—à–Ω—è"]

print(safe_index(fruits, "–ª–∏–º–æ–Ω"))  # -1
```

### `count()` - –ü–æ–¥—Å—á–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –≤—Ö–æ–∂–¥–µ–Ω–∏–π

```python
list.count(value: Any) -> int
```
- **`value`** - –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è –ø–æ–¥—Å—á–µ—Ç–∞

```python

print(count)  # 4

# –ü–æ–¥—Å—á–µ—Ç –≤—Å–µ—Ö —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤
from collections import Counter
numbers = [1, 2, 3, 2, 2, 4, 2]
counter = Counter(numbers)
print(counter)  # Counter({2: 4, 1: 1, 3: 1, 4: 1})

# –ò–ª–∏ –≤—Ä—É—á–Ω—É—é
def count_all(lst):
    counts = {}
    for item in lst:
        counts[item] = counts.get(item, 0) + 1
    return counts

numbers = [1, 2, 3, 2, 2, 4, 2]
print(count_all(numbers))  # {1: 1, 2: 4, 3: 1, 4: 1}
```

words = ["—è–±–ª–æ–∫–æ", "–±–∞–Ω–∞–Ω", "–≤–∏—à–Ω—è", "–∞–±—Ä–∏–∫–æ—Å"]
