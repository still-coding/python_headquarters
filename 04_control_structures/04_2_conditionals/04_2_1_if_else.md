# 🤔 Условный оператор if-else
```toc
minLevel: 2
```
## Что это?

Оператор `if` позволяет выполнять код только при выполнении определённого условия. Это как развилка на дороге — программа выбирает путь в зависимости от условия.

## Синтаксис

### Простой if

```python
if условие:
    # код выполнится, если условие истинно
    действие
```

**Важно:** После `if` ставится двоеточие `:`, а код внутри блока пишется с отступом (обычно 4 пробела).

### Примеры

```python
age = 18

if age >= 18:
    print("Вы совершеннолетний")
```

```python
temperature = 35

if temperature > 30:
    print("Жарко!")
```

## if-else

Когда нужно выполнить одно действие при истинном условии, и другое — при ложном:

```python
if условие:
    # выполнится, если условие истинно
else:
    # выполнится, если условие ложно
```

### Примеры

```python
temperature = 25

if temperature > 30:
    print("Жарко!")
else:
    print("Нормальная температура")
```

```python
password = input("Введите пароль: ")

if password == "secret123":
    print("Доступ разрешён")
else:
    print("Неверный пароль")
```

## if-elif-else

Когда нужно проверить несколько условий:

```python
if условие1:
    # выполнится, если условие1 истинно
elif условие2:
    # выполнится, если условие1 ложно, но условие2 истинно
elif условие3:
    # выполнится, если условие1 и условие2 ложны, но условие3 истинно
else:
    # выполнится, если все условия ложны
```

### Примеры

```python
score = 85

if score >= 90:
    grade = "Отлично"
elif score >= 80:
    grade = "Хорошо"
elif score >= 70:
    grade = "Удовлетворительно"
else:
    grade = "Неудовлетворительно"

print(f"Оценка: {grade}")  # Оценка: Хорошо
```

```python
hour = 14

if hour < 6:
    print("Ночь")
elif hour < 12:
    print("Утро")
elif hour < 18:
    print("День")
else:
    print("Вечер")
# Выведет: День
```

## Вложенные условия

Можно использовать `if` внутри другого `if`:

```python
age = 20
has_license = True

if age >= 18:
    if has_license:
        print("Можете водить машину")
    else:
        print("Нужно получить права")
else:
    print("Вы слишком молоды для вождения")
```

Но часто лучше использовать логические операторы:

```python
age = 20
has_license = True

if age >= 18 and has_license:
    print("Можете водить машину")
elif age >= 18:
    print("Нужно получить права")
else:
    print("Вы слишком молоды для вождения")
```

## Условия и логические операторы

В условиях можно использовать:

### Операторы сравнения
- `==` — равно
- `!=` — не равно
- `>` — больше
- `<` — меньше
- `>=` — больше или равно
- `<=` — меньше или равно

### Логические операторы
- `and` — логическое И (оба условия должны быть истинны)
- `or` — логическое ИЛИ (хотя бы одно условие должно быть истинно)
- `not` — логическое НЕ (инвертирует условие)

```python
age = 25
is_student = True

# and
if age >= 18 and age <= 65:
    print("Трудоспособный возраст")

# or
if age < 18 or age > 65:
    print("Льготная категория")

# not
if not is_student:
    print("Не студент")

# Комбинация
if (age >= 18 and age <= 25) or is_student:
    print("Студенческая скидка доступна")
```

## Что считается истинным и ложным?

Python автоматически преобразует значения в `True` или `False`:

### Ложные значения (False)
- `False`
- `None`
- `0`, `0.0`
- Пустые коллекции: `""`, `[]`, `{}`, `set()`, `()`

### Истинные значения (True)
- Всё остальное: `True`, любое ненулевое число, непустые строки и коллекции

```python
# Проверка на пустоту
name = ""
if name:
    print(f"Привет, {name}!")
else:
    print("Имя не введено")  # Выведет это

# Проверка списка
numbers = []
if numbers:
    print(f"В списке {len(numbers)} элементов")
else:
    print("Список пуст")  # Выведет это
```

## Частые ошибки

### ❌ Присваивание вместо сравнения

```python
x = 5
if x = 5:  # ОШИБКА! Используется = вместо ==
    print("x равен 5")
```

✅ Правильно:
```python
x = 5
if x == 5:
    print("x равен 5")
```

### ❌ Забыть двоеточие

```python
if age >= 18  # ОШИБКА! Нет двоеточия
    print("Совершеннолетний")
```

✅ Правильно:
```python
if age >= 18:
    print("Совершеннолетний")
```

### ❌ Неправильные отступы

```python
if age >= 18:
print("Совершеннолетний")  # ОШИБКА! Нет отступа
```

✅ Правильно:
```python
if age >= 18:
    print("Совершеннолетний")
```
