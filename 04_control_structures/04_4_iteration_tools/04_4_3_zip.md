# 🤐 zip() — объединение последовательностей
```toc
minLevel: 2
```
## Что это?

`zip()` объединяет несколько итерируемых объектов, создавая кортежи из соответствующих элементов. Это как молния на куртке — соединяет элементы параллельно, один к одному.

---

## Синтаксис

```python
zip(*iterables)
```

- `*iterables` — любое количество итерируемых объектов (списки, строки, кортежи и т.д.)

---

## Базовое использование

### Два списка

```python
names = ["Анна", "Борис", "Вера"]
ages = [25, 30, 28]

for name, age in zip(names, ages):
    print(f"{name}: {age} лет")

# Вывод:
# Анна: 25 лет
# Борис: 30 лет
# Вера: 28 лет
```

### Три списка

```python
names = ["Анна", "Борис", "Вера"]
ages = [25, 30, 28]
cities = ["Москва", "СПб", "Казань"]

for name, age, city in zip(names, ages, cities):
    print(f"{name}, {age} лет, живёт в {city}")

# Вывод:
# Анна, 25 лет, живёт в Москва
# Борис, 30 лет, живёт в СПб
# Вера, 28 лет, живёт в Казань
```

---

## Что возвращает zip?

```python
names = ["Анна", "Борис"]
ages = [25, 30]

zipped = zip(names, ages)
print(type(zipped))  # <class 'zip'>

# Преобразование в список
print(list(zipped))
# [('Анна', 25), ('Борис', 30)]
```

**Важно:** `zip()` возвращает итератор, который можно использовать только один раз!

```python
zipped = zip(names, ages)
print(list(zipped))  # [('Анна', 25), ('Борис', 30)]
print(list(zipped))  # [] — уже исчерпан!
```

## Разная длина последовательностей

`zip()` останавливается на самой короткой последовательности:

```python
short = [1, 2]
long = [10, 20, 30, 40]

result = list(zip(short, long))
print(result)  # [(1, 10), (2, 20)]
# Элементы 30 и 40 игнорируются!
```

### Обработка последовательностей разной длины

Если нужно учесть все элементы, используйте `itertools.zip_longest`:

```python
from itertools import zip_longest

short = [1, 2]
long = [10, 20, 30, 40]

result = list(zip_longest(short, long, fillvalue=0))
print(result)
# [(1, 10), (2, 20), (0, 30), (0, 40)]
```

## Распаковка zip (транспонирование)

Оператор `*` может "распаковать" zip обратно:

```python
pairs = [(1, 'a'), (2, 'b'), (3, 'c')]

numbers, letters = zip(*pairs)

print(numbers)  # (1, 2, 3)
print(letters)  # ('a', 'b', 'c')
```

### Транспонирование матрицы

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6]
]

# Транспонируем
transposed = list(zip(*matrix))
print(transposed)
# [(1, 4), (2, 5), (3, 6)]

# Преобразуем обратно в списки
transposed = [list(row) for row in zip(*matrix)]
print(transposed)
# [[1, 4], [2, 5], [3, 6]]
```

## Практические примеры

### Создание словаря

```python
keys = ['name', 'age', 'city']
values = ['Иван', 30, 'Москва']

# Создание словаря из двух списков
person = dict(zip(keys, values))
print(person)
# {'name': 'Иван', 'age': 30, 'city': 'Москва'}
```

### Параллельная обработка списков

```python
prices = [100, 200, 300]
quantities = [2, 1, 3]

# Вычисляем общую стоимость каждого товара
totals = [price * qty for price, qty in zip(prices, quantities)]
print(totals)  # [200, 200, 900]

# Или с циклом
for price, qty in zip(prices, quantities):
    total = price * qty
    print(f"Цена: {price}, количество: {qty}, итого: {total}")
```

### Сравнение списков

```python
expected = [1, 2, 3, 4, 5]
actual = [1, 2, 5, 4, 5]

# Находим различия
differences = [(i, exp, act) for i, (exp, act) in 
               enumerate(zip(expected, actual)) if exp != act]

if differences:
    print("Найдены различия:")
    for i, exp, act in differences:
        print(f"  Позиция {i}: ожидалось {exp}, получено {act}")
else:
    print("Списки идентичны")

# Вывод:
# Найдены различия:
#   Позиция 2: ожидалось 3, получено 5
```

### Объединение данных из нескольких источников

```python
students = ["Анна", "Борис", "Вера"]
math_scores = [85, 92, 78]
physics_scores = [88, 85, 95]

print("Сводка оценок:")
for student, math, physics in zip(students, math_scores, physics_scores):
    average = (math + physics) / 2
    print(f"{student}: математика={math}, физика={physics}, средний балл={average:.1f}")

# Вывод:
# Сводка оценок:
# Анна: математика=85, физика=88, средний балл=86.5
# Борис: математика=92, физика=85, средний балл=88.5
# Вера: математика=78, физика=95, средний балл=86.5
```

### Группировка элементов попарно

```python
data = [1, 2, 3, 4, 5, 6]

# Группируем по парам
pairs = list(zip(data[::2], data[1::2]))
print(pairs)  # [(1, 2), (3, 4), (5, 6)]
```

### Циклический перебор

```python
# Создание бесконечного повторяющегося паттерна
from itertools import cycle

colors = cycle(['красный', 'зелёный', 'синий'])
items = ['яблоко', 'лист', 'небо', 'роза', 'трава']

for item, color in zip(items, colors):
    print(f"{item} → {color}")

# Вывод:
# яблоко → красный
# лист → зелёный
# небо → синий
# роза → красный
# трава → зелёный
```

## Работа с разными типами данных

### Списки и строки

```python
numbers = [1, 2, 3]
letters = "ABC"

for num, letter in zip(numbers, letters):
    print(f"{num} - {letter}")

# Вывод:
# 1 - A
# 2 - B
# 3 - C
```

### Несколько строк

```python
str1 = "Hello"
str2 = "World"

for char1, char2 in zip(str1, str2):
    print(f"{char1} & {char2}")

# Вывод:
# H & W
# e & o
# l & r
# l & l
# o & d
```

### Словари

```python
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict2 = {'a': 10, 'b': 20, 'c': 30}

# Попарное сложение значений
result = {}
for key in dict1:
    result[key] = dict1[key] + dict2[key]

print(result)  # {'a': 11, 'b': 22, 'c': 33}

# Или с zip
for (k1, v1), (k2, v2) in zip(dict1.items(), dict2.items()):
    print(f"{k1}: {v1} + {v2} = {v1 + v2}")
```

## Матричные операции

### Поэлементное сложение

```python
matrix1 = [[1, 2], [3, 4]]
matrix2 = [[5, 6], [7, 8]]

result = [[a + b for a, b in zip(row1, row2)] 
          for row1, row2 in zip(matrix1, matrix2)]

print(result)  # [[6, 8], [10, 12]]
```

### Скалярное произведение векторов

```python
vector1 = [1, 2, 3]
vector2 = [4, 5, 6]

dot_product = sum(a * b for a, b in zip(vector1, vector2))
print(dot_product)  # 1*4 + 2*5 + 3*6 = 32
```

## Комбинирование с другими функциями

### zip + enumerate

```python
names = ["Анна", "Борис", "Вера"]
ages = [25, 30, 28]

for i, (name, age) in enumerate(zip(names, ages), start=1):
    print(f"{i}. {name} ({age} лет)")

# Вывод:
# 1. Анна (25 лет)
# 2. Борис (30 лет)
# 3. Вера (28 лет)
```

### zip + sorted

```python
names = ["Вера", "Анна", "Борис"]
ages = [28, 25, 30]

# Сортируем по имени, сохраняя соответствие возрастов
sorted_data = sorted(zip(names, ages))
sorted_names, sorted_ages = zip(*sorted_data)

print(sorted_names)  # ('Анна', 'Борис', 'Вера')
print(sorted_ages)   # (25, 30, 28)
```

### zip + filter

```python
names = ["Анна", "Борис", "Вера", "Георгий"]
ages = [17, 25, 19, 30]

# Оставляем только совершеннолетних
adults = [(name, age) for name, age in zip(names, ages) if age >= 18]
print(adults)
# [('Борис', 25), ('Вера', 19), ('Георгий', 30)]
```

## Частые ошибки

### ❌ Забыть, что zip останавливается на короткой последовательности

```python
short = [1, 2]
long = [10, 20, 30, 40]

# Ожидаем 4 пары, получаем 2!
result = list(zip(short, long))
print(result)  # [(1, 10), (2, 20)]
```

✅ Используйте `zip_longest`, если нужны все элементы:
```python
from itertools import zip_longest
result = list(zip_longest(short, long, fillvalue=None))
print(result)
# [(1, 10), (2, 20), (None, 30), (None, 40)]
```

### ❌ Попытка повторно использовать zip

```python
zipped = zip([1, 2], ['a', 'b'])

print(list(zipped))  # [(1, 'a'), (2, 'b')]
print(list(zipped))  # [] — пусто!
```

✅ Создавайте новый zip или сохраните результат:
```python
# Вариант 1: создать заново
for item in zip([1, 2], ['a', 'b']):
    print(item)

# Вариант 2: сохранить в список
zipped_list = list(zip([1, 2], ['a', 'b']))
print(zipped_list)  # можно использовать много раз
```

### ❌ Неправильная распаковка

```python
pairs = [(1, 'a'), (2, 'b')]

# ОШИБКА! Пытаемся распаковать в одну переменную
result = zip(*pairs)  # result — это zip-объект, а не кортеж
```

✅ Правильно:
```python
numbers, letters = zip(*pairs)
print(numbers)  # (1, 2)
print(letters)  # ('a', 'b')
```

## Альтернативы zip

### Когда НЕ нужен zip

**Если нужны только индексы:**
```python
# ❌ Бесполезно
for i, _ in zip(range(len(data)), data):
    print(i)

# ✅ Лучше
for i in range(len(data)):
    print(i)
```

**Если нужны индексы И значения:**
```python
# ❌ Сложнее
for i, item in zip(range(len(data)), data):
    print(i, item)

# ✅ Лучше
for i, item in enumerate(data):
    print(i, item)
```

## Полезные паттерны

### Создание пар соседних элементов

```python
numbers = [1, 2, 3, 4, 5]

pairs = list(zip(numbers, numbers[1:]))
print(pairs)
# [(1, 2), (2, 3), (3, 4), (4, 5)]

# Вычисление разностей
differences = [b - a for a, b in zip(numbers, numbers[1:])]
print(differences)  # [1, 1, 1, 1]
```

### Чередование элементов

```python
list1 = [1, 3, 5]
list2 = [2, 4, 6]

# Чередуем элементы
interleaved = [item for pair in zip(list1, list2) for item in pair]
print(interleaved)  # [1, 2, 3, 4, 5, 6]
```

### Свёртка (zip с одним списком)

```python
data = [1, 2, 3, 4]

# Все пары элементов
all_pairs = [(a, b) for a, b in zip(data, data)]
print(all_pairs)
# [(1, 1), (2, 2), (3, 3), (4, 4)]
```
