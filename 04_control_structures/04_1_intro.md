# 🔄 Алгоритмические структуры (управляющие конструкции)

## Что это такое?

Алгоритмические структуры — это инструменты, которые позволяют управлять порядком выполнения кода. Они помогают программе принимать решения и повторять действия.

Позволяют выполнять разный код в зависимости от условий:

- [[04_2_2_ternary|Тернарный оператор]]

### 🔁 Циклы
Позволяют повторять действия:
- [[04_3_1_for_loop|Цикл for]]
- [[04_3_2_while_loop|Цикл while]]
- [[04_3_3_loop_control|Управление циклами (break, continue, else)]]
- [[04_3_4_nested_loops|Вложенные циклы]]

### 🛠️ Инструменты для итерации
Встроенные функции для удобной работы с циклами:
- [[04_4_1_range|range()]]
- [[04_4_2_enumerate|enumerate()]]
- [[04_4_3_zip|zip()]]

### 📦 Включения (Comprehensions)
Компактный способ создания коллекций:
- [[04_5_comprehensions|List/Dict/Set Comprehensions]]

## Зачем это нужно?

Без алгоритмических структур программа выполнялась бы только последовательно, строка за строкой. С их помощью мы можем:
- ✅ Принимать решения на основе данных
- ✅ Повторять действия нужное количество раз
- ✅ Обрабатывать большие объёмы данных
- ✅ Создавать сложную логику программы
