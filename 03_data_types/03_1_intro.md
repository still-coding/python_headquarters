# 📚 Введение в типы данных

## Что такое типы данных?

**Тип данных** — это классификация данных, определяющая:
- Какие значения может принимать переменная
- Какие операции можно выполнять с этими значениями
- Как данные хранятся в памяти компьютера

## Типы данных в Python

### 🗿 Неизменяемые типы (immutable)
Неизменяемые типы данных нельзя изменить после создания. При "изменении" на самом деле создаётся новый объект.

**Основные неизменяемые типы:**
- **Числа**: `int`, `float`, `complex`
- **Строки**: `str`
- **Булевы значения**: `bool`
- **Отсутствие значения**: `NoneType` (значение `None`)
- **Кортежи**: `tuple` (рассматриваются в разделе составных типов)

### ✍️ Изменяемые типы (mutable)
Изменяемые типы данных можно изменять после создания.

**Основные изменяемые типы:**
- **Списки**: `list`
- **Словари**: `dict`
- **Множества**: `set`

## Почему важно понимать типы данных?

1. **Правильные операции**: Каждый тип поддерживает определенные операции
2. **Эффективность**: Разные типы используют память по-разному
3. **Отладка**: Понимание типов помогает находить ошибки
4. **Читаемость кода**: Явное указание типов делает код понятнее

## Проверка типов данных

```python
# Получение типа объекта
type(42)           # <class 'int'>
type("hello")      # <class 'str'>
type(True)         # <class 'bool'>

# Проверка типа
isinstance(42, int)        # True
isinstance("hello", str)   # True
isinstance(3.14, (int, float))  # True (проверка нескольких типов)
```
