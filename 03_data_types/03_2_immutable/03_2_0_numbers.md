## 🧮 Математические операции

```python
# Арифметика
2 + 3      # 5 (сложение)
7 - 2      # 5 (вычитание)
3 * 4      # 12 (умножение)
10 / 3     # 3.3333... (деление)

```

## 🔧 Побитовые операции

> 💡 **Для начинающих:** Побитовые операции — это продвинутая тема, которую можно пропустить при первом изучении. Если ты только начал изучать Python, можешь пока пропустить и перейти сразу к разделу [[#Модуль math|Модуль math →]]

Побитовые операции работают с двоичным представлением чисел на уровне отдельных битов.

### Что такое биты?

Компьютер хранит числа в двоичной системе (только 0 и 1):

```python
# Десятичное число -> двоичное представление
5   # 101   (в двоичной)
7   # 111   (в двоичной)
12  # 1100  (в двоичной)

# Python может показать двоичное представление
bin(5)   # '0b101'
bin(7)   # '0b111'
bin(12)  # '0b1100'
```

### Основные побитовые операции

#### `&` (побитовое И)

Результат = 1, только если оба бита равны 1

```python
5 & 7    # 5 (101 & 111 = 101)
#   101
# & 111
# -----
#   101  = 5

12 & 10  # 8 (1100 & 1010 = 1000)
#   1100
# & 1010
# ------
#   1000  = 8
```

**Практическое применение:**

```python
# Проверка четности (последний бит четного числа всегда 0)
def is_even(n):
    return not n & 1

is_even(4)   # True  (4 & 1 = 0)
is_even(5)   # False (5 & 1 = 1)

# Извлечение младших битов
number = 123
last_4_bits = number & 15  # 15 = 1111 в двоичной
print(f"{number} -> последние 4 бита: {last_4_bits}")  # 123 -> последние 4 бита: 11
```

#### `|` (побитовое ИЛИ)

Результат = 1, если хотя бы один бит равен 1

```python
5 | 7    # 7 (101 | 111 = 111)
#   101
# | 111
# -----
#   111  = 7

12 | 10  # 14 (1100 | 1010 = 1110)
#   1100
# | 1010
# ------
#   1110  = 14
```

**Практическое применение:**

```python
# Установка определенных битов в 1
flags = 0        # 0000
flags |= 1       # Устанавливаем 1-й бит: 0001
flags |= 4       # Устанавливаем 3-й бит: 0101
print(flags)     # 5

# Комбинирование разрешений
READ = 4    # 100
WRITE = 2   # 010
EXECUTE = 1 # 001

permission = READ | WRITE | EXECUTE  # 111 = 7 (все разрешения)
print(f"Полные права: {permission}")  # Полные права: 7
```

#### `^` (побитовое исключающее ИЛИ)

Результат = 1, если биты различны

```python
5 ^ 7    # 2 (101 ^ 111 = 010)
#   101
# ^ 111
# -----
#   010  = 2

12 ^ 10  # 6 (1100 ^ 1010 = 0110)
#   1100
# ^ 1010
# ------
#   0110  = 6
```

**Практическое применение:**

```python
# Простое шифрование (XOR с ключом)
message = 42
key = 123

encrypted = message ^ key
decrypted = encrypted ^ key  # XOR с тем же ключом возвращает исходное
print(f"Исходное: {message}, Зашифрованное: {encrypted}, Расшифрованное: {decrypted}")

# Обмен значений без дополнительной переменной
a = 10
b = 20
a = a ^ b
b = a ^ b  # b = (a ^ b) ^ b = a
a = a ^ b  # a = (a ^ b) ^ a = b
print(f"a = {a}, b = {b}")  # a = 20, b = 10
```

#### `~` (побитовое НЕ / инверсия)

Меняет все биты на противоположные

```python
~5   # -6
# В Python используется дополнительный код для отрицательных чисел
# ~n = -(n + 1)

~0   # -1
~1   # -2
~-1  # 0
```

**Практическое применение:**

```python
# Сброс определенного бита
flags = 7      # 111 (все биты установлены)
flags &= ~2    # Сбрасываем 2-й бит: 111 & ~010 = 111 & 101 = 101
print(flags)   # 5
```

#### `<<` (сдвиг влево)

Сдвигает биты влево, добавляя нули справа

```python
5 << 1   # 10 (101 -> 1010)
5 << 2   # 20 (101 -> 10100)
1 << 3   # 8  (001 -> 1000)
```

**Практическое применение:**

```python
# Быстрое умножение на степени двойки
print(f"7 * 2 = {7 << 1}")    # 7 * 2 = 14
print(f"7 * 4 = {7 << 2}")    # 7 * 4 = 28
print(f"7 * 8 = {7 << 3}")    # 7 * 8 = 56

# Создание степеней двойки
powers_of_2 = [1 << i for i in range(8)]
print(powers_of_2)  # [1, 2, 4, 8, 16, 32, 64, 128]
```

#### `>>` (сдвиг вправо)

Сдвигает биты вправо, отбрасывая младшие биты

```python
10 >> 1  # 5  (1010 -> 101)
20 >> 2  # 5  (10100 -> 101)
8 >> 3   # 1  (1000 -> 1)
```

**Практическое применение:**

```python
# Быстрое деление на степени двойки
print(f"20 / 2 = {20 >> 1}")  # 20 / 2 = 10
print(f"20 / 4 = {20 >> 2}")  # 20 / 4 = 5
print(f"20 / 8 = {20 >> 3}")  # 20 / 8 = 2

# Извлечение старших битов
number = 0b11110000  # 240
upper_4_bits = number >> 4
print(f"Старшие 4 бита числа {number}: {upper_4_bits}")  # Старшие 4 бита числа 240: 15
```

## 🎯 Практические примеры

### Работа с флагами

```python
# Определяем флаги как степени двойки
BOLD = 1      # 0001
ITALIC = 2    # 0010
UNDERLINE = 4 # 0100
STRIKE = 8    # 1000

# Устанавливаем несколько флагов
text_style = BOLD | ITALIC  # 0011

# Проверяем наличие флага
has_bold = (text_style & BOLD) != 0
print(f"Жирный текст: {has_bold}")  # Жирный текст: True

has_underline = (text_style & UNDERLINE) != 0
print(f"Подчеркнутый текст: {has_underline}")  # Подчеркнутый текст: False

# Добавляем флаг
text_style |= UNDERLINE  # 0111
print(f"Стиль с подчеркиванием: {text_style}")  # Стиль с подчеркиванием: 7

# Убираем флаг
text_style &= ~ITALIC  # 0111 & 1101 = 0101
print(f"Стиль без курсива: {text_style}")  # Стиль без курсива: 5
```

### RGB цвета

```python
# RGB цвет как одно 24-битное число
# Красный: биты 16-23, Зеленый: биты 8-15, Синий: биты 0-7

def create_rgb(red, green, blue):
    """Создает RGB цвет из компонентов (0-255)"""
    return (red << 16) | (green << 8) | blue

def extract_rgb(color):
    """Извлекает RGB компоненты из цвета"""
    red = (color >> 16) & 0xFF    # Сдвигаем вправо на 16, берем младшие 8 бит
    green = (color >> 8) & 0xFF   # Сдвигаем вправо на 8, берем младшие 8 бит
    blue = color & 0xFF           # Берем младшие 8 бит
    return red, green, blue

# Создаем фиолетовый цвет (255, 0, 255)
purple = create_rgb(255, 0, 255)
print(f"Фиолетовый цвет: {purple}")  # Фиолетовый цвет: 16711935
print(f"В шестнадцатеричном: 0x{purple:06X}")  # В шестнадцатеричном: 0xFF00FF

# Извлекаем компоненты
r, g, b = extract_rgb(purple)
print(f"RGB: ({r}, {g}, {b})")  # RGB: (255, 0, 255)
```

## 💡 Полезные трюки

```python
# Проверка, является ли число степенью двойки
def is_power_of_2(n):
    return n > 0 and not (n & (n - 1))

print(is_power_of_2(8))   # True  (8 = 2³)
print(is_power_of_2(6))   # False

# Округление до ближайшей степени двойки
def next_power_of_2(n):
    if n <= 0:
        return 1
    return 1 << (n - 1).bit_length()

print(next_power_of_2(10))  # 16
print(next_power_of_2(17))  # 32

# Подсчет количества единичных битов

# Способ 1: Проверяем каждый бит (интуитивный)
def count_bits_simple(n):
    count = 0
    while n:
        count += n & 1  # Проверяем последний бит
        n >>= 1         # Сдвигаем вправо на 1 позицию
    return count

# Способ 2: Алгоритм Брайана Кернигана (оптимальный)
def count_bits_fast(n):
    count = 0
    while n:
        count += 1
        n &= n - 1  # Убираем самый правый единичный бит
    return count

# Проверяем оба способа
print(f"Простой способ: {count_bits_simple(7)}")   # 3 (111 содержит 3 единицы)
print(f"Быстрый способ: {count_bits_fast(7)}")     # 3

# Демонстрация разницы в эффективности
print(f"Число 1024 (только 1 единица в старшем разряде):")
print(f"Простой способ обработает {1024.bit_length()} битов")  # 11 битов
print(f"Быстрый способ обработает {bin(1024).count('1')} единицу")  # 1 единицу

# Объяснение трюка n & (n-1):
# Пример с числом 12 (1100):
# 12 в двоичном:     1100
# 12 - 1 = 11:       1011
# 12 & 11:           1000  <- убрали самую правую единицу!
#
# Следующая итерация:
# 8 в двоичном:      1000
# 8 - 1 = 7:         0111
# 8 & 7:             0000  <- убрали последнюю единицу!
```

> **Важно:** Побитовые операции чаще используются в системном программировании, работе с аппаратурой или оптимизации. В повседневном программировании на Python они нужны реже, но понимание их работы полезно для общего развития.

## Модуль `math`

```python
import math

math.sqrt(16)       # 4.0 (квадратный корень)
math.sin(math.pi / 2) # 1.0 (синус 90°)
math.log(math.e)    # 1.0 (натуральный логарифм)
math.factorial(5)   # 120 (факториал)

from math import sin, cos, pi
sin(pi) ** 2 + cos(pi) ** 2 # 1.0
```

## 🔎 Проверка на ходу

Интерактивный REPL (Read-Evaluate-Print Loop) режим - твой друг!
Используй его, чтобы быстро проверять какие-то гипотезы, делать промежуточные проверки и вычисления. 

```python
>>> type(5 + 3.14)
<class 'float'>  # результат будет float, если хотя бы один операнд float

>>> "AB" * 3
'ABABAB'  # строка повторяется 3 раза

>>> 5 / 2
2.5  # результат деления - float, даже если оба операнда целые

>>> 7 // 2
3  # целочисленное деление отбрасывает дробную часть
```

## 🎯 Полезные встроенные функции для чисел

```python
# Минимум и максимум
min(1, 5, 3)       # 1
max(1, 5, 3)       # 5

# Сумма (для чисел)
sum([1, 2, 3, 4])  # 10

# Абсолютное значение
abs(-5)            # 5
abs(3.14)          # 3.14

# Округление
round(3.14159)     # 3
round(3.14159, 2)  # 3.14 (до 2 знаков после запятой)
```
